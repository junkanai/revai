## How to use
### grow AI
You can compile related files.

`$ make main`

Then, you can run the cpp program.

`$ ./main`


### play Reversi vs AI
You can play if you have already installed pygame.

`$ python3 main.py`

If you haven't, you can easily install pygame.

`$ pip3 install pygame`

Have fun!
